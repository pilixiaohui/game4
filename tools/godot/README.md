# Godot binary

`bin/godot` is the project entrypoint used by automation.

The Linux Godot executable is not committed as a normal Git blob because the
current binary is larger than GitHub's 100 MB limit. For a fully portable clone,
place `Godot_v4.7-stable_linux.x86_64` in this directory or install `godot4` /
`godot` on `PATH`.
